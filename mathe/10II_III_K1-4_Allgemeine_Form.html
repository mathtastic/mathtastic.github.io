Aufgabe 1 — Vermutungs-Markt: Was bedeutet das c?
a) + b) Richtig ist Karte B: „c ist der y-Achsenabschnitt“.
Begründung: Wenn man x = 0 in y = ax² + bx + c einsetzt, fallen die beiden ersten Terme weg (0·a + 0·b = 0) und es bleibt nur y = c. Also schneidet die Parabel die y-Achse bei (0|c).
Karte A ist falsch (b ist die Steigung der Parabel an x = 0, kein Achsenabschnitt). Karte C ist falsch (a ist der Streckfaktor aus K1-2). Karte D ist falsch — nur bei c = 0 ginge die Parabel durch (0|0).

Aufgabe 3 — Erfinde dein Beispiel
Mögliche Beispiel-Funktion (offen, viele Lösungen):
a) z.B. y = 2x² + 5 (b = 0)
b) Einsetzen x = 0: y =2·0² + 5 = 5. Die Parabel schneidet die y-Achse bei (0|5).
c) Da b = 0, ist die Parabel symmetrisch zur y-Achse. Der Scheitelpunkt liegt damit AUF der y-Achse — bei S(0|5).
Generell gilt: bei b = 0 ist xS = 0 und yS = c. Der Scheitel ist also (0|c).
Aufgabe 4 — Schnelltest: y-Achsenabschnitt
a) y = 5x² − 3  →  y-Achsenabschnitt = −3 (c-Wert direkt ablesen)
b) y = −2x² + 7  →  y-Achsenabschnitt = 7
c) Parabel durch (0|−4)  →  c = −4 (denn y = c (wenn x = 0), hier y =−4)
Merksatz — Allgemeine Form
In der Allgemeinen Form y = ax² + bx + c ist c der y-Achsenabschnitt der Parabel. Setzt man x = 0 ein, ergibt sich y =c. Wenn b = 0 ist, liegt der Scheitelpunkt auf der y-Achse.
