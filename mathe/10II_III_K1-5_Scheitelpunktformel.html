Scheitelpunktformel — direkt aus der Allgemeinen Form
❖ Knobel-Start
Bestimme den Scheitelpunkt von f: y = x² + 6x + 5. 2 Minuten überlegen — wie würdest du vorgehen? Wertetabelle? Etwas aus Klasse 8? Schau, ob du den Scheitelpunkt findest, BEVOR du weiterliest.
Schwierig, oder? Mit der quadratischen Ergänzung aus Klasse 8 kommst du zum Ziel, aber es dauert. In deiner Formelsammlung steht eine schnellere Methode: die Scheitelpunktformel. Damit berechnest du den Scheitelpunkt direkt aus a, b, c — in wenigen Sekunden.
Heute: Tempoduett. Erst die Formel verstehen (Aufgabe 1), dann Block A alleine. Wer fertig ist, sucht einen Partner und korrigiert gemeinsam. Dann Block B.
Allgemeine Form
f: y = a x² + b x + c

f: y = a (x − xS)² + yS

Scheitelpunktformel — direkt aus der Allgemeinen Form — Lösung
Aufgabe 1 — Formel anwenden Schritt für Schritt
a) f: y = x² + 6x + 5 → a = 1, b = 6, c = 5
b) xS = −b2a = −62·1 = −3
c) yS = c − b²4a = 5 − 364 = 5 − 9 = −4
→ Scheitelpunkt S(−3|−4)
Aufgabe 2 — Block A: Routinefälle
a) y = x² + 4x − 1: a=1, b=4, c=−1 → xS = −42 = −2, yS = −1 − 164 = −1 − 4 = −5 → S(−2|−5)
b) y = x² − 6x + 11: xS = 62 = 3, yS = 11 − 364 = 11 − 9 = 2 → S(3|2)
c) y = x² + 8x + 10: xS = −82 = −4, yS = 10 − 644 = 10 − 16 = −6 → S(−4|−6)
d) y = x² − 10x + 18: xS = 102 = 5, yS = 18 − 1004 = 18 − 25 = −7 → S(5|−7)

Aufgabe 3 — Block B: Vorsicht beim a
a) y = 2x² + 8x + 7: a=2, b=8 → xS = −84 = −2, yS = 7 − 648 = 7 − 8 = −1 → S(−2|−1)
b) y = −x² + 4x + 1: a=−1, b=4 → xS = −4−2 = 2, yS = 1 − 16−4 = 1 + 4 = 5 → S(2|5) (nach unten geöffnet, Hochpunkt!)
c) y = 0,5x² − 2x + 5: a=0,5, b=−2 → xS = 21 = 2, yS = 5 − 42 = 5 − 2 = 3 → S(2|3)
d) y = −2x² + 12x − 13: a=−2, b=12 → xS = −12−4 = 3, yS = −13 − 144−8 = −13 + 18 = 5 → S(3|5)
