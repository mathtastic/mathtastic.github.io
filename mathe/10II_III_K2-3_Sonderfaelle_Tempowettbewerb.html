Rein- und gemischtquadratische Gleichungen
Heute Tempo-Duell. Die Mitternachtsformel kennst du aus der letzten Stunde — sie löst jede quadratische Gleichung. Aber: bei zwei Sonder­formen ist sie der Umweg.
Heute lernst du, wann du sie weglassen kannst. Pro Block: halbe Klasse rechnet mit Formel, halbe Klasse mit Trick — Zeit messen, vergleichen, Mehrwert erkennen.

Aufgabe 2 — Ausklammern: x² + bx = 0
a)Der zweite Sonderfall: x² + bx = 0 (kein c). Klammere x aus (Distributivgesetz rückwärts) — dann Satz vom Nullprodukt: ein Produkt ist Null, wenn ein Faktor Null ist. Damit hast du ohne Mitternachtsformel die Lösungen.

x² + 6x = 0
x ausklammern → Nullprodukt
b
x² − 8x = 0
x ausklammern → Nullprodukt
c
2x² + 10x = 0
2x ausklammern → Nullprodukt
d
−x² + 4x = 0
−x ausklammern → Nullprodukt

Aufgabe 1 — 60-Sek-Duell
Tandem A (Formel): a=1, b=0, c=−25 → D = 0 + 100 = 100 → x1,2 = ±102 = ±5. ~5–6 Rechen-Schritte.
Tandem B (Wurzel): x = ±√25 = ±5. 1 Schritt.
b) Wurzelziehen ist 3–5× schneller, weil bei x² = a die Formel mit b = 0 läuft — die meisten Schritte sind dann sinnlos.
Aufgabe 2 — Ausklammern (x² + bx = 0)
a) x² + 6x = 0 → x(x+6) = 0 → x1 = 0, x2 = −6
b) x² − 8x = 0 → x(x−8) = 0 → x1 = 0, x2 = 8
c) 2x² + 10x = 0 → 2x(x+5) = 0 → x1 = 0, x2 = −5
d) −x² + 4x = 0 → −x(x−4) = 0 → x1 = 0, x2 = 4
Bei gemischt ohne c ist x = 0 immer eine Lösung — das sieht man auch ohne Formel sofort.
Aufgabe 3 — Optimal-Wahl + Lösungen
a) x² = 100 → W → x = ±10
b) x² + 7x + 12 = 0 → M → D = 49 − 48 = 1 → x1,2 = −7 ± 12 → x1 = −3, x2 = −4
c) x² − 9x = 0 → A → x(x − 9) = 0 → x1 = 0, x2 = 9
d) 5x² = 80 → x² = 16 → W → x = ±4
e) 2x² + 5x − 3 = 0 → M → D = 25 + 24 = 49 → x1,2 = −5 ± 74 → x1 = 0,5; x2 = −3
f) −3x² + 12x = 0 → A → −3x(x − 4) = 0 → x1 = 0, x2 = 4
Faustregel: b = 0 → W · c = 0 → A · sonst → M.
