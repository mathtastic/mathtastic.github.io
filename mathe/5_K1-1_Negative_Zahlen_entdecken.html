Negative Zahlen im Alltag — Lösung
Aufgabe 1 — Stimmt das wirklich?
Alle vier Behauptungen stimmen! Überall gibt es „weniger als nichts": a) −5 °C im Winter · b) Konto im Minus (Schulden) · c) Tiefgarage U1, U2 · d) Minuspunkte beim Spiel. Individuelle Beispiele der Schüler.
Aufgabe 2 — Zahl mit Vorzeichen
7 °C über null = +7 · 4 °C unter null = −4 · Guthaben 20 € = +20 · Schulden 15 € = −15 · 3 Stockwerke tiefer = −3 · 6 m Tauchtiefe = −6.
Aufgabe 3 — Verbinde (mehrere Linien möglich)
−6 → ist eine negative Zahl. +4 → ist eine positive Zahl & liegt rechts von der 0. 0 → ist weder positiv noch negativ. −2 → ist eine negative Zahl & passt zu 2 °C unter null.
Merksatz
… negative Zahlen … positive … kleiner als 0 … links von der 0 … ganzen Zahlen.
