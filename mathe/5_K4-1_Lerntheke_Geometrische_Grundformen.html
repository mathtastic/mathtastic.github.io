Aufgabe 3 — Gold: Schrägbilder (maßstabsgetreu, 2 Kästchen = 1 cm)
Würfel a = b = c = 4 cm · Tiefenkante halb so lang: 2 cm (45°)
Quader a = 5, b = 3, c = 2 cm · Tiefenkante 3 cm → 1,5 cm (45°)
Vorderfläche a × c in echter Größe; die Breite b unter dem Verzerrungswinkel 45° und mit q = 0,5: die schräge Tiefenkante wird halb so lang gezeichnet — Würfel b = 4 cm → 2 cm, Quader b = 3 cm → 1,5 cm (mit Lineal/Geodreieck unter 45° abtragen). Verdeckte Kanten gestrichelt. c) Meister: gleichschenklig-rechtwinkliges Dreieck (zwei gleiche Katheten am rechten Winkel); gleichschenkliges Trapez (zwei parallele Seiten, beide Schenkel gleich lang).
⚠ Typischer Stolperstein: Alle drei schrägen Breite-Kanten gleich lang, parallel und halbiert (q = 0,5) — und die verdeckte Kante gestrichelt.
⚡ Bonus für Schnelle
Zeichne das Schrägbild eines dreiseitigen Prismas (Vorderfläche = Dreieck) mit Tiefe b = 3 cm.
Tipp: Ein Prisma hat zwei gleiche Grundflächen — hier zwei Dreiecke.
Bonus-Lösung — kopfüber, erst selbst knobeln! 🔄
Dreieck in echter Größe; von jeder Ecke eine schräge Kante unter 45°, halbiert (b = 3 cm → 3 Kästchen); Endpunkte zum hinteren Dreieck verbinden, verdeckte Kanten gestrichelt.
