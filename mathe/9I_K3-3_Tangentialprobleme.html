Tangentialprobleme — wann ist eine Gerade Tangente?
Heute: Tangenten finden. Aus K2-4 weißt du: D = 0 → Tangente. Heute drehen wir die Frage um: für welches t ist y = mx + t Tangente an die Parabel y = x²?
Plus heute: drei Mitschüler streiten über Tangenten (Aufg 3) — und eine KI behauptet etwas, das du überprüfen sollst (Aufg 4).
Aufgabe 1 — Tangenten-Bedingung mit Parameter t
a)Wir suchen eine Tangente an y = x² mit Steigung m = 2. Die Geradenschar lautet y = 2x + t (mit Parameter t). Setze Parabel und Gerade gleich, bringe auf Normalform, und finde den Wert von t, bei dem die Diskriminante D = 0 wird.
a
Gegeben: y = x²  und  y = 2x + t  —  gesucht: t mit D = 0
Aufgabe 2 — Tangente an y = x² mit Steigung 4
a)Jetzt selbst: Finde den Wert von t, sodass y = 4x + t Tangente an y = x² ist. Bestimme anschließend den Berührpunkt (Tangenten-Punkt).
a
Gegeben: y = x²  und  y = 4x + t  —  gesucht: t und Berührpunkt

Tangentialprobleme — wann ist eine Gerade Tangente? — Lösung
Aufgabe 1 — Tangenten-Bedingung y = 2x + t
Gleichsetzen: x² = 2x + t → x² − 2x − t = 0
D = (−2)² − 4·1·(−t) = 4 + 4t
Tangente: D = 0 → 4 + 4t = 0 → t = −1
→ Tangentengleichung: y = 2x − 1 (vgl. K2-4 Aufg. A!)
Berührpunkt: x = 1 (Doppellösung) → S(1|1).
Aufgabe 2 — Tangente y = 4x + t
Gleichsetzen: x² = 4x + t → x² − 4x − t = 0
D = 16 + 4t = 0 → t = −4
→ Tangente: y = 4x − 4
Berührpunkt: x = 4/2 = 2 → y = 4·2 − 4 = 4 → S(2|4).
Aufgabe 3 — Concept Cartoon
Anna ✓ stimmt: y = 0 (x-Achse) und y = x² gleichgesetzt → x² = 0 → Doppellösung x = 0, also D = 0 → Tangente im Scheitel (0|0).
Ben ✓ stimmt: y = c (c > 0) und y = x² → x² = c → x = ±√c (zwei Lösungen, D > 0) → Sekante.
Cleo ✗ stimmt NICHT: auch schräge Geraden können Tangenten sein — siehe Aufg 1 (y = 2x − 1) und Aufg 2 (y = 4x − 4). Tangente hängt von D = 0 ab, nicht von der Steigung.
Aufgabe 4 — KI-Werkstatt: y = 6x − 9 Tangente?
①  Punkt (3|9) prüfen: y = x² → 3² = 9 ✓ · y = 6·3 − 9 = 9 ✓ → liegt auf beiden Kurven.
②  Gleichsetzen: x² = 6x − 9 → x² − 6x + 9 = 0
③  Diskriminante: D = 36 − 36 = 0 → doppelte Lösung x = 3 → ja, D ist 0.
④  KI hat recht! y = 6x − 9 berührt y = x² genau im Punkt (3|9). Lehre: KI-Antworten können stimmen — aber prüfen ist Pflicht.
